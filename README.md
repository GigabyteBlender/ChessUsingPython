# Chess Game

A fully-featured chess game built with Python and Pygame, featuring a clean MVC architecture.

## Features

- Complete chess rules implementation (castling, en passant, pawn promotion)
- Move validation with check and checkmate detection
- Move history in algebraic notation
- Captured pieces display with material advantage
- Board rotation after each move
- Visual highlights for legal moves and captures

## Requirements

- Python 3.7+
- Pygame

## Installation

1. Clone the repository
2. Install dependencies:
```bash
pip install pygame
```

## How to Run

Run the game from the project root directory:

```bash
python code/chess.py
```

## Controls

- **Click** to select and move pieces
- **F** - Toggle fullscreen
- **R** - Restart game
- **ESC** - Exit to menu/quit
